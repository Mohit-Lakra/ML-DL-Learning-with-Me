# Linear Combinations, Span, and Basis Vectors

## Linear Combinations
A linear combination of a set of vectors is an expression made up of those vectors multiplied by scalars (which can be any real numbers) and then added together. For example, if we have two vectors **v** and **w**, a linear combination of these vectors would look like:
```a * v + b * w
```
where `a` and `b` are scalars.

## Span
The span of a set of vectors is the collection of all possible linear combinations of those vectors. For example, if we have two vectors **v** and **w**, the span of these vectors would include all vectors that can be expressed as:
```a * v + b * w
```
for any scalars `a` and `b`. The span can be thought of as the "space" that is covered by the linear combinations of the vectors. If the span of a set of vectors fills the entire space (like a plane or 3D space), we say that the vectors are "spanning" that space.

## Basis Vectors
A basis of a vector space is a set of vectors that are linearly independent and span the entire space. This means that any vector in the space can be expressed as a linear combination of the basis vectors, and no vector in the basis can be written as a linear combination of the others.
For example, in a 2D space, the standard basis vectors are:
- **i** = (1, 0)
- **j** = (0, 1)
These vectors are linearly independent and span the entire 2D space, meaning any vector in 2D can be expressed as a linear combination of **i** and **j**. In a 3D space, the standard basis vectors are:
- **i** = (1, 0, 0)
- **j** = (0, 1, 0)
- **k** = (0, 0, 1)
These three vectors are linearly independent and span the entire 3D space.
