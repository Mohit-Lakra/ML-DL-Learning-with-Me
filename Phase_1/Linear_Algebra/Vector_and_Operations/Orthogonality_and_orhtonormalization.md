# Orthogonality & Orthonormalization
Two vectors are orthogonal if their dot product is zero ($\mathbf{u} \cdot \mathbf{v} = 0$). This means they are at a 90° angle to each other.

## Orthogonal Sets
A set of vectors $\{\mathbf{v}_1, \dots, \mathbf{v}_k\}$ is an orthogonal set if every possible pair of different vectors in the set is orthogonal.

## Orthonormal Sets
A set is orthonormal if:
1. It is an orthogonal.
2. Every vector has a unit length ($\|\mathbf{v}\| = 1$).
