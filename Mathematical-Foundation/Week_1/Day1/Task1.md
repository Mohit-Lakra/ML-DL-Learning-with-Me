## Vector addition 
- Geometric interpretation :- Place the tail of the second vector at the head of the first vector, then draw a vector from the tail of the first vector to the head of the second vector. This new vector is the sum of the two vectors.

- Algebraic interpretation :- Add the corresponding components of the vectors together. For example, if we have two vectors A = (a1, a2) and B = (b1, b2), then their sum C = A + B would be calculated as follows: C = (a1 + b1, a2 + b2).

## Scalar Multiplication
- Geometric interpretation :- Scaling a vector by a scalar changes its magnitude but not its direction. For example, if we have a vector A and we multiply it by a scalar k, the resulting vector B = kA will have a magnitude that is k times the magnitude of A, but it will point in the same direction as A if k is positive, and in the opposite direction if k is negative.
